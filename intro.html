Introductory Python Programs

Build instructions for intro programs

Linux command line:

Option 1: Call the interpreter
$ python 'filename'.py or $ python3 'filename'.py

Option 2: Let the script call the interpreter
1. Make sure the first line of your file has #!/usr/bin/env python
2. Make the file executable - chmod +x 'filename'.py
3. Then execute the program as ./'filename'.py

Example 1:

This program will roll a random 6 sided dice and print the result:

import random #imports the module which allows for the generation of random numbers>>

print("Rolling a dice.") #simple text output to notify the user what we're doing
print(random.randint(1,6)) #printing a randomly generated integer between 1 and 6, to represent the 6 sides of a regular dice

Example 2:

This program takes in a number and computes the summation of the given number.

num = input("Please enter a number\n") #input is the function that allows the program to take data from stdin
num = int(num) #cast the given input string to an integer
count = 0
for j in range(num+1): #iterates from 0 to n
count = count + j #update our count variable
print(count) #output count to screen
